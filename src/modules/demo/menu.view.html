Menus

Les menus sont gérés via le module sidemenu

Il suffit de créer un élément avec les classes side-menu et side-menu-left (ou right suivant le sens d'ouverture), ainsi que l'animation souhaitée

Pour le moment nous ne gérons qu'un seul menu, une future version gérera plusieurs menus en simultané sur les pages, ainsi que de nouvelles animations
