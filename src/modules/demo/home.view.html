Bienvenue dans l'application de démo

La cordova fuse contient de base:

  • Cordova configuré pour Angular
  • Une structure pour les modules et pages
  • Ui-router, ng-animate, et notre module ng-transitions pour faire tous les effets de transition souhaités
  • Notre module sideMenu pour gérer les menus gauche et droite, avec différents effets d'apparition
